Method

The model initializes Gaussian centers from occupied voxels, uses non-saturating occupancy aggregation, and trains with sparse slice supervision plus sampled 3D occupancy points. The current run is intentionally small: it fits one subject directly to produce a clean reconstruction mesh and measurable artifacts. 

python3 src_code/scripts/fit_subject_reconstruction.py --run-dir runs/subject_fit_v01 --split val --sample-index 0 --num-gaussians 1800 --num-samples 4096 --steps 600
