Method

The model initializes Gaussian centers from occupied voxels, uses non-saturating occupancy aggregation, and trains with sparse slice supervision plus sampled 3D occupancy points. The current run is intentionally small: it verifies that the reconstruction path produces a measurable checkpoint and viewable artifacts. 

python3 src_code/scripts/train.py --epochs 1 --checkpoint-every 1 --run-dir runs/reconstruction_working_v01 --batch-size 1 --num-samples 512 --num-gaussians 500 --image-loss-weight 0.05 --sparse-loss-weight 0.01
